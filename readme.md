Version 1.0.0


### Install

```bash
# Clone the repo
git clone git@github.com:LesliTech/lesli.dev.git

# Install ruby dependencies
bundle install

# Install npm dependencies
npm install

# Run the dev server
middleman server

# Open the website on your browser
http://localhost:4567

```
