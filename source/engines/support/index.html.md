### Quick start

```shell
# Add LesliSupport engine gem
bundle add lesli_support
```

```shell
# Setup & initialize the database
rake lesli:db:setup
```

```ruby
# Load LesliSupport engine
Rails.application.routes.draw do
    mount LesliSupport::Engine => "/support"
end
```
