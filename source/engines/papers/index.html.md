### Quick start

```shell
# Add LesliPapers engine gem
bundle add lesli_papers
```

```shell
# Setup & initialize the database
rake lesli:db:setup
```

```ruby
# Load LesliPapers engine
Rails.application.routes.draw do
    mount LesliPapers::Engine => "/papers"
end
```
