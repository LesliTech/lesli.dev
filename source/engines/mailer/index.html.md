### Quick start

```shell
# Add LesliMailer engine gem
bundle add lesli_mailer
```

```shell
# Setup & initialize the database
rake lesli:db:setup
```

```ruby
# Load LesliMailer engine
Rails.application.routes.draw do
    mount LesliMailer::Engine => "/mailer"
end
```
