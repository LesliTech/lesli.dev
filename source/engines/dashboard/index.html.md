### Quick start

```shell
# Add LesliDashboard engine
bundle add lesli_dashboard
```

```shell
# Setup database
rake lesli:db:setup
```

```ruby
# Load LesliDashboard
Rails.application.routes.draw do
    mount LesliDashboard::Engine => "/dashboard"
end
```
